# Hanflow Site

Official website (Landing + Docs) for Hanflow, a fusion agent framework with privacy routing, RAG, and observability.

Pure frontend Next.js 14 static site. Deploy to Vercel in one click.

## Develop

```bash
npm install
npm run dev
```

Open http://localhost:3000 (redirects to /en).

## Stack

Next.js 14 App Router, TypeScript, Tailwind CSS, next-intl, next-mdx-remote, Shiki, Motion, Geist.

## Content

Docs live under `content/<version>/<locale>/` as MDX. UI strings live in `messages/<locale>.json`.
